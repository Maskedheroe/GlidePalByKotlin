# GlidePalByKotlin
A util for using Glide easily, made it by kotlin

USE IT !

Add it in your root build.gradle at the end of repositories:
allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
  
 
THEN :
 Add the dependency
 dependencies {
	        implementation 'com.github.Maskedheroe:GlidePalByKotlin:v3.0'
}
